# Política de Segurança

## Versões Suportadas

Atualmente, apenas a versão mais recente deste projeto recebe atualizações de segurança.

| Versão | Suportada          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Reportando uma Vulnerabilidade

Nós levamos a segurança deste projeto a sério. Se você descobrir uma vulnerabilidade de segurança, por favor, siga estas etapas:

1.  **NÃO** abra uma Issue pública no GitHub.
2.  Envie um e-mail com os detalhes para: **machado.brunob@gmail.com**.
3.  Inclua passos para reproduzir a falha, se possível.

Nós tentaremos responder em até 48 horas para confirmar o recebimento e discutir os próximos passos.

Agradecemos sua ajuda em manter este projeto seguro para todos!
